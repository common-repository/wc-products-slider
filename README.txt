== Description ==
Make your Woo store more gorgeous and attractive with advanced 3D Slider!
This simple and easy user friendly plugin will allow you to show your products categories with a 3D slider and your products that will bring more sale without any extra effort!

Core Features-

* Category Slider
1. 3D sliders
2. 3 types of slides design (themes) are available
3. Simple and Easy to use.
4. Auto slide with nav bar
5. Only nav bar
6. Auto slider without nav bar

* Features of Products Slider
1. 4 Layouts with different Design
2. Responsive and Mordern design
3. Customize color, font-size, hover effect
4. Show / Hide product title, Price of each slider
5. Show / Hide view more button of each slider
6. Show / Hide 'Add to cart' button of each slider
7. Change/override 'add to cart' button and 'view more' button text
8. Easy Control from Admin Panel
9. New layout ( Total = 5 layouts )
10.Live Control preview


* How to use
1. Navigate to WC Products Slider menu and copy shortcode (choose anyone from 3)
2. Paste the shortcode on any page and done!

* Fully compatible with [Elementor](https://elementor.com), Gutenberg, Classic Editor.


== Installation ==

1. Upload the plugin folder after extracting it to the "/wp-content/plugins/(the folder of the extracted plugin)" directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Now navigate to WC Products Slider menu and copy shortcode (choose anyone from 3)
4. Paste the shortcode on any page and done!
